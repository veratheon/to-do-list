# My todo list

Реализовано на react с использованием хуков useState, useEffect.  
Файлы загружаются на firebase с возможностью скачивания или просмотра.  
Хостится на firebase.  
Код откомментирован с помощью JSDoc. 

## Возможные функции

Добавление задач.  
Удаление задач.  
Редактирование.  
Пометка "выполнено".
